Langkah Menggunakan Aplikasi

  1. Buka menu Transaksi.
  2. Isi form transaksi (Nama Transaksi, Jumlah, Kategori).
  3. Klik tombol Simpan untuk menyimpan transaksi.
  4. Klik link Daftar Transaksi untuk melihat data dalam tabel.
  5. Menu Profil berisi deskripsi aplikasi.
  6. Menu Login berisi form login.

Info

Data transaksi disimpan menggunakan localStorage (tanpa database) agar tetap bisa menampilkan daftar transaksi.
